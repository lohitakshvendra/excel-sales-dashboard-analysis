📊 Excel Sales Dashboard Analysis

An interactive Microsoft Excel dashboard built using Superstore Sales Data to analyze sales performance, profitability, and business trends.

---

 📌 Project Overview

This project demonstrates Excel data analysis skills using:

- Pivot Tables
- Pivot Charts
- Slicers
- Timeline Filter
- Conditional Formatting
- Interactive Dashboard Design

The dashboard helps answer business questions such as:

- What is the total sales?
- Which category generates the highest profit?
- How are sales changing over time?
- Which products or categories perform best and worst?

📈 Key Insights

- Total Sales are displayed dynamically.
- Technology generated the highest sales.
- Furniture had comparatively lower profit.
- Sales increased during peak shopping seasons.
- Conditional formatting quickly identifies top and bottom performers.
